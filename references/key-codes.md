# macOS Key Codes 速查

| Key Code | 按键 | 用途 |
|----------|------|------|
| 36 | Return | 发送消息/确认选择 |
| 48 | Tab | 切换焦点 |
| 49 | Space | 空格 |
| 51 | Delete | 删除（退格） |
| 53 | Escape | 取消/关闭 |
| 116 | Page Up | 向上翻页 |
| 121 | Page Down | 向下翻页 |
| 123 | Left Arrow | 左箭头 |
| 124 | Right Arrow | 右箭头 |
| 125 | Down Arrow | 下箭头 |
| 126 | Up Arrow | 上箭头 |

# 微信 macOS 快捷键

| 快捷键 | 功能 |
|--------|------|
| Cmd+N | 发起会话（新建群聊，不是搜索！） |
| Cmd+F | 全局搜索（包含网络搜索，不推荐用于查找会话） |
| Cmd+A | 全选 |
| Cmd+V | 粘贴 |

# 微信 UI 布局（相对于窗口）

```
+------------------------------------------+
|  搜索栏 (x+120, y+33)                     |
+--------+---------------------------------+
|        |                                 |
| 会话   |         聊天区域                  |
| 列表   |   (x+200+(w-200)/2, y+h/2)      |
| ~200px |                                 |
|        |                                 |
|        +---------------------------------+
|        |      消息输入框                   |
|        | (x+200+(w-200)/2, y+h-55)       |
+--------+---------------------------------+
```

# 注意事项

- `keystroke` 不支持中文/emoji，必须使用剪贴板粘贴
- 搜索栏是左侧列表顶部的搜索框，不是 Cmd+F
- Cmd+F 会打开全局搜索（含网络搜索），不适合查找会话
- 搜索回车后可能有残留文字在输入框，需要 Cmd+A + Delete 清理
- UI 自动化需要辅助功能权限（终端 + claude CLI）
